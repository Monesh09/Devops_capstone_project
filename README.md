# DevOps Capstone Project: Node.js Web Application

## 📌 Project Description
This project demonstrates an end-to-end DevOps pipeline for a Node.js web application using GitHub, Jenkins, Docker, AWS EC2, Prometheus, Grafana, and automation with Bash + Cron.

## 🛠 Tech Stack
- Source Control: Git + GitHub
- CI/CD: Jenkins on EC2
- App: Node.js (Express)
- Containerization: Docker + Docker Hub
- Infra: AWS EC2 (Ubuntu)
- Monitoring: Prometheus, Grafana, Node Exporter
- Automation: Bash + Cron jobs
